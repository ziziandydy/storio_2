## ADDED Requirements

### Requirement: Immersive Scrolling UI
The landing page SHALL implement a dark-themed, immersive scrolling experience using Folio Black (#0d0d0d) and Storio Gold (#c5a059).

#### Scenario: Visual Reveal on Scroll
- **WHEN** the user scrolls down the page
- **THEN** sections (Features, Team, CTA) SHALL fade in and slide up smoothly.

### Requirement: Sticky Glassmorphic Header
The page SHALL have a sticky top navigation bar with a glassmorphism effect (blur background).

#### Scenario: Navigation Access
- **WHEN** the user is anywhere on the page
- **THEN** the header SHALL remain at the top with a visible "Explore Storio" button.
