## Requirements
### Requirement: Feature Showcase
The landing page SHALL highlight the three core features of Storio 2: Memory Timeline (Rewatch), Visual Templates (Sharing), and Bottom-Focused Search.

#### Scenario: Explaining Rewatch Philosophy
- **WHEN** the user views the Memory Timeline section
- **THEN** the content SHALL explain that each viewing creates a unique memory card.

### Requirement: Cross-Platform Guidance
The page SHALL clearly state platform availability for iOS, Desktop Web, and Mobile Web.

#### Scenario: Platform CTAs
- **WHEN** the user views the Hero or Footer section
- **THEN** they SHALL see distinct "Try on Mobile" and "Try on Web" buttons.
